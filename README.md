# dotfiles-rs

reads my dotfiles repository and does stuff

## Features

* `jokeyrhyme-dotfiles sync`:

  * copies or creates symbolic links from `~/.dotfiles` into `~` for settings

  * otherwise creates and/or edits settings files in `~`
